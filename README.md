# Green Grants Platform

A web-based platform that helps individuals and contractors identify and access eligible green energy subsidies efficiently.

## Problem
Navigating green energy grants is fragmented, confusing, and time-consuming for homeowners and small contractors.

## Solution
This platform centralizes subsidy data, matches users to relevant programs, and streamlines decision-making through a simple web interface.

## Features
- Grant and subsidy database
- Contractor resource matching
- User-friendly web interface
- Built with Flask and SQLite

## Tech Stack
- Python (Flask)
- SQLite
- HTML/CSS
- CSV data processing

## How to Run
```bash
pip install -r requirements.txt
python app.py
